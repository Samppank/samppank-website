Discord webhook -lähetys (Python)

Lähettää yksinkertaisen tekstiviestin Discord-kanavalle webhookin kautta. Vaihda WEBHOOK_URL omaan webhook-osoitteeseesi.

# Discord webhook example (Python)
# Asenna tarvittaessa: pip install requests
import requests
import json

WEBHOOK_URL = "https://discord.com/api/webhooks/PUT_YOUR_WEBHOOK_HERE"  # korvaa omallasi

def send_webhook(content="Hei! Tämä on testiviesti Samppankilta"):
    data = {
        "content": content,
        "username": "Samppank Bot",
        "embeds": [
            {
                "title": "Ilmoitus",
                "description": "Tämä viesti lähetettiin webhookilla.",
                "color": 5814783
            }
        ]
    }
    headers = {"Content-Type": "application/json"}
    resp = requests.post(WEBHOOK_URL, data=json.dumps(data), headers=headers)
    if resp.status_code == 204 or resp.status_code == 200:
        print("Webhook lähetetty onnistuneesti.")
    else:
        print(f"Virhe: {resp.status_code} - {resp.text}")

if __name__ == '__main__':
    send_webhook("Testiviesti: webhook toimii!")

Numeron arvaus -peli (JavaScript)

Pelaaja arvaa satunnaisen numeron 1–100 saaden vihjeen suurempi/pienempi.

// Number guess game (JavaScript)
(function(){
  let secret = Math.floor(Math.random()*100) + 1;
  let attempts = 0;
  console.log("Aloita arvaaminen (1-100). Käytä guess(n) funktiota konsolissa.");

  window.guess = function(n){
    attempts++;
    if(n === secret){
      console.log(`Oikein! Löysit numeron ${secret} ${attempts} yrityksellä.`);
      // uusi kierros
      secret = Math.floor(Math.random()*100) + 1;
      attempts = 0;
      console.log("Uusi numero luotu. Arvaa taas!");
    } else if(n < secret){
      console.log("Liian pieni.");
    } else {
      console.log("Liian suuri.");
    }
  };
})();

Ajan näyttö -peli (Stop the second)

Pelaajan tehtävänä on pysäyttää ajannäyttö kun sekunnit näyttävät tiettyä lukua — testi tarkasta ajantajua.

// Clock game (JavaScript) - Stop the second
// Tämä UI-esimerkki voidaan liittää HTML:ään: nappi "Start"/"Stop" ja tekstikenttä tulokselle.
(function(){
  let running = false;
  let interval;
  let target = Math.floor(Math.random() * 60); // tavoitteena saada tämä sekunti
  console.log("Tavoiteltu sekunti:", target);

  window.startClockGame = function(){
    if(running) return;
    running = true;
    interval = setInterval(() => {
      const d = new Date();
      process.stdout && process.stdout.write ? null : null;
      // Konsoli-demo: näyttää sekunnin - käytä selainta ja DOM:ia parempaan UI:hin
      console.log("Sekunti:", d.getSeconds());
    }, 800);
    console.log("Peli käynnissä. soita stopClockGame() pysäyttääksesi.");
  };

  window.stopClockGame = function(){
    if(!running) return;
    clearInterval(interval);
    running = false;
    const sec = new Date().getSeconds();
    console.log("Pysäytit sekunnin:", sec);
    if(sec === target){
      console.log("Voitto — osuit täsmälleen!");
    } else {
      console.log("Ei osunut. Tavoite oli:", target);
    }
    target = Math.floor(Math.random() * 60);
    console.log("Uusi tavoite:", target);
  };
})();

Tiedostojen luontiesimerkki (Batch & Python)

Esimerkit luovat 100 tyhjää .txt -tiedostoa paikalliseen kansioon. Älä aja mitään mitä et ymmärrä — nämä tiedostot luodaan vain jos suoritat itse skriptin omalla koneellasi.

@echo off
for /l %%i in (1,1,100) do (
    echo. > tiedosto_%%i.txt
)
echo Luotu 100 tyhjää tekstitiedostoa!
pause
